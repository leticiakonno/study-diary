# 🚀 Tech Study Diary & Career Transition

Bem-vindo(a) ao meu repositório de estudos! Este espaço foi criado para documentar a minha jornada de transição de carreira para a área de TI e Dev, além de servir como um guia open source para outros desenvolvedores em início de carreira.

Trocar de carreira é sempre um trabalho difícil, principalmente quando são áreas distintas, mas sigo buscando progresso todos os dias para aprimorar meus conhecimentos e me manter firme no mercado! 

## 🎯 Sobre o Projeto
Acredito que compartilhar conhecimento é a melhor forma de fixar o que aprendemos. Aqui, registro diariamente meus avanços, desafios de lógica, projetos práticos de bootcamps e anotações sobre desenvolvimento.

Minha transição envolve trazer uma forte base analítica (da Engenharia) para a resolução de problemas lógicos em desenvolvimento de software.

## 🛠️ Tecnologias e Linguagens Abordadas
Neste repositório, você encontrará códigos e estudos envolvendo, **Front-End**, **Back-End**, **Banco de Dados**, **Infraestrutura e Redes**, **Lógica de Programação** e muito mais! 

## 📂 Como navegar
O repositório está dividido por áreas de estudo. Sinta-se à vontade para explorar as pastas.

## 🤝 Como contribuir
Este é um projeto **Open Source**! Se você tem uma solução mais otimizada para algum desafio, ou quer adicionar um artigo sobre transição de carreira, sua ajuda é muito bem-vinda.
1. Faça um *Fork* do projeto
2. Crie uma *Branch* para sua modificação (`git checkout -b feature/MinhaContribuicao`)
3. Faça o *Commit* (`git commit -m 'Adicionando resolução em Python para o desafio X'`)
4. Faça o *Push* (`git push origin feature/MinhaContribuicao`)
5. Abra um *Pull Request*
